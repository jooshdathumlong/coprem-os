# 🧠 COPREM: Skill Playbooks (SOP Database)

> **"We don't make the same mistake twice. We don't learn the same lesson twice."**

This folder stores the accumulated knowledge and Standard Operating Procedures (SOPs) of the team.

## How it works (The Evolution Loop):
1. A team member completes a task and discovers a winning formula (e.g., a specific prompt structure, a perfect copy formula, a successful trading setup).
2. The team member creates a Playbook file here (e.g., `job_email_framework.md` or `personal_trading_checklist.md`).
3. Next time Jeff delegates a similar task, he instructs the team: **"Read the playbook first."**
4. The team executes perfectly on the first try.
