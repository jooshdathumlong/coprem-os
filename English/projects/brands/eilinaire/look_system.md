# EILINAIRE PRODUCT LOOK SYSTEM
## Single-Axis Garment Operating System

> Source: Prem (founder) | Stored by Jeff | 2026-05-28

## 0. System Definition

A garment system that functions as a **"behavioral operating system of the body."** Garments are designed from **Body State Logic**, not fashion or trends. Clothing is a **"decision mechanism"**, not decoration.

> **1 garment = 1 axis only (Single Axis Only)**

| Axis | Body Logic |
|------|-----------| 
| **FOCUS** | Control Structure — body control system |
| **CALM** | Soft Yield — body resistance release system |
| **PRESENT** | Expression Surface — identity expression through texture |
| **RESET** | Void Identity — perceptual identity erasure system |

**No cross-axis mixing in any single product.**

## 1. Core Philosophy

The body is a **"status display instrument of the mind."**

| Axis | Body Mode |
|------|----------|
| FOCUS | Body is controlled and organized |
| CALM | Body is released to relax |
| PRESENT | Body is used as expression surface |
| RESET | Body's perception is reduced to near-invisibility |

> *"What must the body undergo in this state?"* If the answer is more than 1 = product violates the system.

## 2. Axis Product System

### 2.1 FOCUS — THE ARMOR SYSTEM
**Function:** Control body structure — lock form to be still, sharp, precise.

| Parameter | Spec |
|-----------|------|
| Products | Tailored shirts, suits, blazers, slacks |
| Fabric | Poly-Wool Blend / High Twist |
| GSM | 250–400 |
| Fit | Structured — precise shoulder, waist, leg lock |
| Seam | Clean structural + high-precision edge control |
| Hardware | All hidden — nothing draws the eye |
| Surface | Smooth, sharp, matte, clear geometry |
| Color | Deep, still tones — black, dark grey, navy, dark olive |
| Forbidden | Draping fabrics / soft details / uncontrollable textures |

### 2.2 CALM — THE SHIELD SYSTEM
**Function:** Reduce body resistance — feel safe, light, unrestricted.

| Parameter | Spec |
|-----------|------|
| Products | Soft tees, drape knits, flow pants, soft layer covers |
| Fabric | Bamboo Modal / Tencel Blend |
| GSM | 140–190 |
| Fit | Relaxed drape — let gravity control the form |
| Seam | Hidden / soft overlock — minimal friction |
| Hardware | Blended with fabric — no sound, no reflection |
| Surface | Continuous soft flow, no sharp edges |
| Color | Soft, muted, low contrast — fog grey, cream, sand |
| Forbidden | Rigid structure / suits / skin-resistant materials |

### 2.3 PRESENT — THE TACTILE SYSTEM
**Function:** Create visual awareness — draw attention through texture and surface expression.

| Parameter | Spec |
|-----------|------|
| Products | Patterned garments, 3D knit, jacquard, statement jackets |
| Fabric | Raw Linen Blend / 3D Knit / Jacquard |
| GSM | 200–240 |
| Fit | Balanced expressive — not lost in looseness, not controlled by tightness |
| Seam | Visible craft — topstitch 1/4"–3/8", visible craftsmanship |
| Hardware | Level 2 — visible, part of design |
| Surface | Dimensional, textured, highly expressive |
| Color | Vivid with contrast — emerald, vivid blue, orange |
| Forbidden | Flat/identity-less / disappearing minimal / FOCUS structure |

### 2.4 RESET — THE CLOUD SYSTEM
**Function:** Erase fashion identity — body no longer perceived as a social object.

| Parameter | Spec |
|-----------|------|
| Products | Sleepwear, tank tops, air tees, home wear, underwear system |
| Fabric | Airy Gauze / Muslin |
| GSM | < 130 |
| Fit | Zero compression — no forcing, no shaping |
| Seam | Flatlock — minimize all seams |
| Hardware | Level -1 — none |
| Surface | Like "air" — no weight, no fashion identity |
| Color | Bright — white, pearl, cool light tones |
| Forbidden | Outerwear / statement / any structure |

## 3. Cross-Axis Layering

Mixing only via **wearing layers externally**. Each product maintains 100% axis purity.

```
Outer:    FOCUS or PRESENT
Mid:      CALM or PRESENT
Base:     RESET or CALM
Private:  RESET only
```

## 4. Manufacturing Spec System

Every product defined by **6 axis-locked parameters:** Fabric Type, GSM Range, Fit Logic, Seam System, Hardware Level, Texture Rule. **No borrowing from other axes.**

## 5. Hard Constraint Engine

1. 1 product = 1 axis only
2. No cross-axis material/technique borrowing
3. No hybrid products
4. Mixing only via layering
5. RESET restricted to private spaces — never as outer identity

## 6. Decision Engine

Before design/production, answer: (1) Body must be controlled/relaxed/expressed/erased? (2) Body forced or freed? (3) What should the viewer feel?

> **If answer is unclear — do NOT produce.**

## 7. Summary

> Not fashion. Not style. It is **decision architecture of the human body through garments.**
