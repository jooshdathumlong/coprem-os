> **"เราจะไม่ทำผิดพลาดซ้ำสอง และเราจะไม่เรียนรู้บทเรียนซ้ำสองเช่นกัน"**

โฟลเดอร์นี้เก็บข้อมูลความรู้ที่สะสมมาและขั้นตอนการปฏิบัติงานที่เป็นมาตรฐาน (SOPs) ของทีม

## วิธีการทำงาน (วงจรการพัฒนาทักษะ):
1. สมาชิกในทีมทำงานชิ้นหนึ่งเสร็จสิ้นและค้นพบสูตรสำเร็จ (เช่น โครงสร้างคำสั่งที่สมบูรณ์แบบ, สูตรการเขียนคำโฆษณาที่ได้ผลดี, รูปแบบการเทรดที่ประสบความสำเร็จ)
2. สมาชิกในทีมสร้างไฟล์คู่มือ (Playbook) ไว้ที่นี่ (เช่น `job_email_framework.md` หรือ `personal_trading_checklist.md`)
3. ในครั้งต่อไปที่ Jeff มอบหมายงานที่คล้ายกัน เขาจะสั่งการทีมงานว่า: **"โปรดอ่านคู่มือ (Playbook) ก่อนเริ่มงาน"**
4. ทีมงานจะปฏิบัติงานได้อย่างถูกต้องสมบูรณ์แบบตั้งแต่การพยายามครั้งแรก

*(ปัจจุบันยังว่างอยู่ คู่มือจะถูกเพิ่มเข้ามาเรื่อยๆ ระหว่างการทำงาน)*
