# ปลั๊กอินท่อส่งข้อมูล (Data Pipelines Plugin)
(ระบบเสริมขีดความสามารถภายใน)

**ขีดความสามารถ:**
- การแยกวิเคราะห์ไฟล์ CSV และการรันคำสั่ง SQL โดยอัตโนมัติ
- การจัดระเบียบกระบวนการ ETL (Extract, Transform, Load) สำหรับคลังข้อมูล (Data Warehouses)
- การเชื่อมต่อแหล่งข้อมูลที่กระจัดกระจายเพื่อสร้างรายงานแบบรวมศูนย์
