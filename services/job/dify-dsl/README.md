# Dify DSL — Agent Import Files

วิธี import agent เข้า Dify:
1. เข้า Dify → Studio → Create App → Import DSL File
2. เลือกไฟล์ `.yml` ที่ต้องการ
3. หลัง import → ไปที่ Settings → ตั้ง LLM model → Publish
