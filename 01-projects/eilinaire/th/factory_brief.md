## 1. PURPOSE OF THIS DOCUMENT

เอกสารนี้กำหนด:
- โครงสร้างสินค้าแบบแยกสภาวะ (FOCUS / CALM / PRESENT / RESET)
- มาตรฐานการผลิตของแต่ละแกน
- ข้อห้ามเชิงโครงสร้างที่โรงงานต้องปฏิบัติตาม
- รูปแบบการตีความสินค้าแบบ "1 สินค้า = 1 สภาวะ"
- มาตรฐานคุณภาพระดับ production line

---

## 2. CORE PRODUCTION PRINCIPLE

| Rule | Detail |
|------|--------|
| **1 SKU = 1 Axis** | เสื้อผ้า 1 SKU = 1 แกนสภาวะเท่านั้น |
| **No mixing** | ห้ามผสมคุณสมบัติข้ามแกนในสินค้าเดียว |
| **Layering only** | การผสมแกนทำได้เฉพาะ "การสวมหลายชั้น" |
| **System, not fashion** | โรงงานต้องตีความสินค้าแบบ "ระบบ" ไม่ใช่ "แฟชั่น" |

> หากมีการตีความแบบผสม จะถือว่า "สินค้าผิดระบบ" และต้องผลิตใหม่

## 3. AXIS PRODUCTION STANDARD

### 3.1 FOCUS — The Armor System
**เป้าหมาย:** ทำให้โครงร่างชัดเจน ล็อกทรงให้คงที่ สร้างภาพลักษณ์แข็งแรงและนิ่ง

| Spec | Requirement |
|------|------------|
| Fabric | Poly-Wool Blend หรือผ้าทอแน่น |
| GSM | 250–400 |
| Surface | เรียบ แน่น ไม่ยืดตัวมาก |
| Pattern | ทรงเข้ารูปชัดเจน ไหล่คมและนิ่ง ไม่มีทิ้งตัว |
| Seam | Structural seam เก็บขอบแบบคม SPI สูง |
| Hardware | ซ่อนทั้งหมด ห้ามโชว์โลหะ |

**ข้อห้ามผลิต:**
- ❌ ห้ามใช้ผ้านิ่มลื่นแบบ CALM
- ❌ ห้ามใช้ทรงหลวม
- ❌ ห้ามมี texture แบบ PRESENT

---

### 3.2 CALM — The Soft Shield System
**เป้าหมาย:** ให้ร่างกายผ่อนคลาย ลดแรงกดทับ ให้ผ้าทิ้งตัวตามธรรมชาติ

| Spec | Requirement |
|------|------------|
| Fabric | Bamboo / Modal / Tencel |
| GSM | 140–190 |
| Surface | นุ่ม ลื่น ระบายอากาศดี |
| Pattern | ทรงหลวมพอดี ไม่บังคับรูปทรง รองรับการเคลื่อนไหว |
| Seam | ตะเข็บซ่อนหรือ overlock นุ่ม ลดการเสียดสี |
| Hardware | กลืนไปกับผ้า ห้ามโลหะเด่น |

**ข้อห้ามผลิต:**
- ❌ ห้ามโครงสร้างสูท
- ❌ ห้ามผ้าแข็ง
- ❌ ห้าม detail ที่ดึงสายตา

---

### 3.3 PRESENT — The Expression System
**เป้าหมาย:** ดึงสายตา แสดง texture และ detail สร้างความโดดเด่น

| Spec | Requirement |
|------|------------|
| Fabric | Linen blend / 3D knit / jacquard |
| GSM | 200–240 |
| Surface | มีมิติผิวชัดเจน |
| Pattern | กึ่งเข้ารูป มี visual layering เน้นจุดโฟกัส |
| Seam | Visible stitching / topstitch ชัดเจน งาน craft ต้องเห็น |
| Hardware | อนุญาตระดับโชว์ (metal / decorative) |

**ข้อห้ามผลิต:**
- ❌ ห้ามเรียบจนไม่มีตัวตน
- ❌ ห้าม minimal แบบ CALM
- ❌ ห้ามซ่อนทุกอย่างแบบ FOCUS

---

### 3.4 RESET — The Void System
**เป้าหมาย:** เบาที่สุด ไร้โครงสร้าง ไร้ความเป็นแฟชั่น

| Spec | Requirement |
|------|------------|
| Fabric | Muslin / airy gauze / light cotton |
| GSM | < 130 |
| Surface | โปร่ง เบา ระบายอากาศสูง |
| Pattern | Oversized มาก ไม่มี shaping ไม่กำหนดสรีระ |
| Seam | Flatlock เท่านั้น ลดรอยต่อให้มากที่สุด |
| Hardware | ห้ามมีทั้งหมด |

**ข้อห้ามผลิต:**
- ❌ ห้ามใช้เป็น outerwear
- ❌ ห้ามมีดีเทลดึงสายตา
- ❌ ห้ามโครงสร้างทุกชนิด

## 4. CROSS-AXIS CONTROL RULE

โรงงานต้องปฏิบัติดังนี้:

| Rule | Detail |
|------|--------|
| No material mixing | ห้ามผสมวัสดุข้ามแกนใน SKU เดียว |
| No pattern reuse | ห้ามใช้ pattern ของอีกแกนมา reuse |
| No middle ground | ห้าม "ตีความกลาง" ระหว่าง 2 แกน |
| Single axis per SKU | ทุก SKU ต้อง assign axis เดียวเท่านั้น |

**ตัวอย่างผิดระบบ:**
- ❌ CALM fabric + FOCUS structure
- ❌ PRESENT hardware + CALM silhouette

---

## 5. LAYERING SYSTEM

โรงงานไม่ต้องผลิต "ชุดผสม" — User จะประกอบเองภายหลัง

```
Outer Layer:  FOCUS or PRESENT
Mid Layer:    CALM or PRESENT
Base Layer:   RESET or CALM
Private:      RESET only
```

**หลักสำคัญ:** สินค้าแต่ละชิ้น "บริสุทธิ์" — การผสมเกิดนอกโรงงานเท่านั้น

---

## 6. QUALITY CONTROL LOGIC

ก่อนปล่อยสินค้า ต้องตรวจ:

| Check | Criteria |
|-------|---------|
| Single Axis | SKU นี้มี 1 axis เท่านั้น |
| No Cross-axis | ไม่มีการแอบใช้ attribute ข้ามแกน |
| Spec Compliance | fabric / fit / seam / hardware ตรง axis |
| Style Clarity | ไม่มีความกำกวมเชิงสไตล์ |

> **ถ้าข้อใด "ไม่ชัด" = REJECT**

## 7. PRODUCTION INTENT

โรงงานต้องเข้าใจว่า:

> นี่ไม่ใช่การผลิตแฟชั่นทั่วไป

| Element | Meaning |
|---------|---------|
| เสื้อ | ระบบตัดสินใจของร่างกาย |
| ผ้า | ตัวกำหนดสภาวะ |
| โครงสร้าง | ตัวกำหนดพฤติกรรม |
