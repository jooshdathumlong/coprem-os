# EILINAIRE PRODUCT LOOK SYSTEM
## Single-Axis Garment Operating System

> Source: เปรม (founder) | Stored by Jeff | 2026-05-28

---

## 0. SYSTEM DEFINITION

EILINAIRE PRODUCT LOOK SYSTEM คือระบบการสร้างสินค้าเสื้อผ้าที่ทำงานเหมือน **"ระบบปฏิบัติการเชิงพฤติกรรมของร่างกาย"**

- เสื้อผ้าไม่ถูกออกแบบจากแฟชั่นหรือเทรนด์ แต่ถูกออกแบบจาก **"สถานะของร่างกาย (Body State Logic)"**
- ระบบนี้ถือว่าเสื้อผ้าเป็น **"กลไกการตัดสินใจ"** ไม่ใช่ของตกแต่ง

### CORE PRINCIPLE
> **เสื้อผ้า 1 ชิ้น ต้องมีเพียง 1 แกนสภาวะ (Single Axis Only)**

| Axis | Body Logic |
|------|-----------|
| **FOCUS** | Control Structure — ระบบควบคุมร่างกาย |
| **CALM** | Soft Yield System — ระบบผ่อนแรงต้านของร่างกาย |
| **PRESENT** | Expression Surface System — ระบบการแสดงตัวตนผ่านผิวสัมผัส |
| **RESET** | Void Identity System — ระบบการลบตัวตนออกจากการรับรู้ |

**ห้ามผสมแกนในสินค้าเดียวทุกกรณี**

---

## 1. CORE PHILOSOPHY

ระบบนี้มองว่า **ร่างกายคือ "เครื่องมือแสดงสถานะของจิตใจ"**

| Axis | Body Mode |
|------|----------|
| FOCUS | ร่างกายถูกควบคุมและจัดระเบียบ |
| CALM | ร่างกายถูกปล่อยให้ผ่อนคลาย |
| PRESENT | ร่างกายถูกใช้เป็นพื้นที่แสดงออก |
| RESET | ร่างกายถูกลดทอนการรับรู้จนเกือบหายไป |

**คำถามเดียวที่ทุกสินค้าต้องตอบ:**
> *"ร่างกายในสภาวะนี้ต้องถูกทำอะไร"*
> ถ้าคำตอบมากกว่า 1 = สินค้านั้นผิดระบบ

## 2. AXIS PRODUCT SYSTEM

### 2.1 FOCUS — THE ARMOR SYSTEM
**หน้าที่:** ควบคุมโครงสร้างร่างกาย — ล็อกรูปทรงให้นิ่ง คม ชัดเจน

| Parameter | Spec |
|-----------|------|
| **Products** | เสื้อเชิ้ตเทเลอร์, เสื้อสูท, เบลเซอร์, กางเกงสแล็ค |
| **Fabric** | Poly-Wool Blend / High Twist Fabric |
| **GSM** | 250–400 |
| **Fit** | Structured fit — ล็อกไหล่ เอว และโครงขาอย่างแม่นยำ |
| **Seam** | Clean structural seam + edge control ความแม่นยำสูง |
| **Hardware** | ซ่อนทั้งหมด — ไม่มีสิ่งใดดึงสายตา |
| **Surface** | เรียบ คม ด้าน มี geometry ชัดเจน |
| **Color** | โทนเข้ม ลึก นิ่ง — ดำ เทาเข้ม กรมท่า เขียวมะกอกเข้ม |
| **Forbidden** | ผ้าทิ้งตัว / ดีเทลอ่อนนุ่ม / texture ที่ควบคุมทรงไม่ได้ |

---

### 2.2 CALM — THE SHIELD SYSTEM
**หน้าที่:** ลดแรงต้านของร่างกาย — ให้ร่างกายรู้สึกปลอดภัย เบา ไม่ถูกบังคับ

| Parameter | Spec |
|-----------|------|
| **Products** | เสื้อยืดผ้านุ่ม, เสื้อถักทิ้งตัว, กางเกงผ้า flow, เสื้อคลุม soft layer |
| **Fabric** | Bamboo Modal / Tencel Blend |
| **GSM** | 140–190 |
| **Fit** | Relaxed drape fit — ปล่อยให้แรงโน้มถ่วงควบคุมทรง |
| **Seam** | Hidden seam / soft overlock — ลดแรงเสียดทาน |
| **Hardware** | กลืนไปกับผ้า — ไม่มีเสียง ไม่มีการสะท้อน |
| **Surface** | Soft flow ต่อเนื่อง ไม่มีขอบคม |
| **Color** | โทนอ่อน หม่น low contrast — เทาหมอก ครีม ทราย |
| **Forbidden** | โครงสร้างแข็ง / สูท / วัสดุที่สร้างแรงต้านต่อผิว |

---

### 2.3 PRESENT — THE TACTILE SYSTEM
**หน้าที่:** สร้างการรับรู้ผ่านสายตา — ดึง attention ผ่าน texture และ surface expression

| Parameter | Spec |
|-----------|------|
| **Products** | เสื้อผ้าที่มีลาย, 3D knit, เสื้อ jacquard, แจ็กเก็ต statement |
| **Fabric** | Raw Linen Blend / 3D Knit / Jacquard |
| **GSM** | 200–240 |
| **Fit** | Balanced expressive fit — ไม่หลวมจนหาย ไม่แน่นจนควบคุม |
| **Seam** | Visible craft — topstitch 1/4"–3/8" มองเห็นงานฝีมือ |
| **Hardware** | Level 2 — สามารถเห็นและเป็นส่วนหนึ่งของ design |
| **Surface** | มีมิติ มี texture มีความ expressive สูง |
| **Color** | สดและมี contrast — เขียวมรกต น้ำเงินสด ส้ม |
| **Forbidden** | ความเรียบจนไม่มีตัวตน / minimal ที่กลืนหาย / โครงสร้าง FOCUS |

---

### 2.4 RESET — THE CLOUD SYSTEM
**หน้าที่:** ลบตัวตนทางแฟชั่น — ทำให้ร่างกายไม่ถูกมองว่าเป็น object ทางสังคม

| Parameter | Spec |
|-----------|------|
| **Products** | ชุดนอน, เสื้อกล้าม, เสื้อ air tee, เสื้อใส่อยู่บ้าน, underwear system |
| **Fabric** | Airy Gauze / Muslin |
| **GSM** | < 130 |
| **Fit** | Zero compression fit — ไม่มีการบังคับ ไม่มีการ shaping |
| **Seam** | Flatlock — ลดรอยต่อให้มากที่สุด |
| **Hardware** | Level -1 — ไม่มีทั้งหมด |
| **Surface** | เหมือน "อากาศ" — ไม่มีความหนัก ไม่มีตัวตนทางแฟชั่น |
| **Color** | สว่าง — ขาว ขาวมุก โทนเย็นอ่อน |
| **Forbidden** | Outerwear / statement / โครงสร้างทุกชนิด |

## 3. CROSS-AXIS LAYERING SYSTEM

การผสมสภาวะสามารถเกิดขึ้นได้เฉพาะ **"ภายนอกสินค้า" ผ่านการสวมทับเท่านั้น**
สินค้าทุกชิ้นต้องคงความบริสุทธิ์ของแกนตัวเอง 100%

```
Layer       Axis Options
───────────────────────────────
Outer       FOCUS or PRESENT
Mid         CALM or PRESENT
Base        RESET or CALM
Private     RESET only
```

| Role | Axis |
|------|------|
| FOCUS | ควบคุมภาพรวมของ look |
| CALM | เป็น comfort buffer |
| PRESENT | เป็น focal expression |
| RESET | ใช้เฉพาะระดับส่วนตัวเท่านั้น |

---

## 4. MANUFACTURING SPEC SYSTEM

ทุกสินค้าต้องถูกกำหนดผ่าน **6 parameters ที่ล็อกตามแกน:**

1. ชนิดผ้า (Fabric Type)
2. น้ำหนักผ้า (GSM Range)
3. รูปแบบ fit (Fit Logic)
4. ระบบตะเข็บ (Seam System)
5. ระดับฮาร์ดแวร์ (Hardware Level)
6. กฎพื้นผิว (Texture Rule)

**ทุก parameter ต้องถูกกำหนดโดยแกนเดียว — ห้าม borrow จากแกนอื่น**

---

## 5. HARD CONSTRAINT ENGINE

| # | Rule |
|---|------|
| 1 | สินค้า 1 ชิ้น = 1 แกนเท่านั้น |
| 2 | ห้ามยืมวัสดุหรือเทคนิคข้ามแกน |
| 3 | ห้ามสร้าง hybrid product ทุกกรณี |
| 4 | การผสมทำได้เฉพาะผ่าน layering เท่านั้น |
| 5 | RESET ต้องจำกัดเฉพาะพื้นที่ส่วนตัว ห้ามใช้เป็น outer identity |

## 6. DECISION ENGINE

ก่อนออกแบบหรือผลิตสินค้า ต้องตอบคำถาม:

1. ร่างกายต้องถูก **ควบคุม / ผ่อนคลาย / แสดงออก / ลบออก**?
2. ร่างกายควรถูก **บังคับ หรือปล่อยอิสระ**?
3. ภาพที่เกิดขึ้นควรทำให้ผู้มองรู้สึกอะไร?

> **ถ้าคำตอบไม่ชัดเจน — ห้ามผลิตสินค้า**
