# 👥 EGO ERA — Character Boxsets

โฟลเดอร์นี้เก็บข้อมูลเจาะลึกของตัวละครแต่ละตัว (Boxset) ซึ่งแยกเป็นโฟลเดอร์ย่อยตามชื่อตัวละคร

## 📂 โครงสร้างโฟลเดอร์ตัวละคร
- `rin/` - ริน (Anchor)
- `jin/` - จิน (Trigger)
- `ray/` - เรย์ (Rupture)
- `ken/` - เคน (Shadow)
- `pie/` - พาย (Healer)
- `guy/` - กาย (Absorber)
- `ann/` - แอน (Shield/Safe Space)
- `jean/` - จีน (Mirror)
- `so/` - โซ (Warden)
- `bomb/` - บอม (Spark)
- `pao/` - เปา (Relief)
- `nay/` - เน (Relief)

แต่ละโฟลเดอร์สามารถเก็บ Profile, Reference, และพัฒนาการของตัวละครแต่ละตัวแยกจากกัน เพื่อความสะดวกในการค้นหาและอ้างอิง
