# 📈 กลยุทธ์การเทรดและกฎควบคุมความเสี่ยง (Trading Strategy & Risk Rules - สำหรับเตรียมสอบ The5ers)

> **"การเทรดไม่ใช่เรื่องของการเป็นคนถูก แต่เป็นเรื่องของการบริหารความเสี่ยงอย่างเข้มงวดและเล่นกับสถิติความน่าจะเป็น"**

---

## 🛡️ กฎการควบคุมความเสี่ยง (Risk Management Rules - ห้ามละเมิดเด็ดขาด)

1. **ความเสี่ยงสูงสุดต่อการเทรด 1 ครั้ง (Max Risk Per Trade):** จำกัดไว้ที่ `0.5%` ถึง `1.0%` ของเงินทุนเริ่มต้นบัญชี ห้ามมีข้อยกเว้นเด็ดขาด
2. **ขีดจำกัดการขาดทุนสูงสุดรายวัน (Max Daily Loss Limit):** ห้ามขาดทุนเกิน `2%` ของพอร์ตต่อวัน (หากขาดทุนถึงจุดนี้ ให้ปิดหน้าจอและหยุดเทรดทันทีในวันนั้น)
3. **ขีดจำกัดการลากพอร์ตสูงสุด (Max Drawdown Limit - สำหรับกองทุน):** ปฏิบัติตามเกณฑ์ควบคุมความเสี่ยงของกองทุน The5ers อย่างเคร่งครัด (โดยปกติจำกัดการลากพอร์ตอยู่ที่ 5-10% ขึ้นอยู่กับรูปแบบการสอบ)
4. **ช่วงเวลาเทรด (Trading Hours):** จำกัดเวลาเทรดเฉพาะในช่วงที่มีสภาพคล่องและปริมาณซื้อขายสูง เช่น ช่วงเปิดตลาดลอนดอน (London Session) หรือนิวยอร์ก (NY Session) เพื่อลดผลกระทบจากค่าสเปรด (Spread)
5. **ห้ามเทรดล้างแค้น (No Revenge Trading):** หากแพ้ติดต่อกัน 2 ครั้ง ให้หยุดพักจากการเทรดเป็นเวลาอย่างน้อย 4 ชั่วโมงทันทีเพื่อปรับสภาวะจิตใจ

## ⚙️ การตั้งค่าแผนกลยุทธ์ (Strategy Setup - รายการตรวจสอบความได้เปรียบ)

### 1. บริบทตลาดและแนวโน้มหลัก (Market Context / Trend - Daily/H4)
- [ ] แนวโน้มในกรอบเวลาใหญ่ (Timeframe) คืออะไร? (ขาขึ้น Bullish / ขาลง Bearish / ไซด์เวย์ออกข้าง Range)
- [ ] กำหนดจุดหรือโซนสภาพคล่องหลัก (Liquidity Zones) แล้วหรือยัง? (แนวรับ-แนวต้าน Support/Resistance / บล็อกคำสั่งซื้อขาย Order Blocks)

### 2. จุดชนวนสำหรับเข้าซื้อขาย (Entry Triggers - H1/M15)
- [ ] มีการกวาดสภาพคล่อง (Liquidity Sweep) หรือการยืนยันการเปลี่ยนโครงสร้างราคา (Change of Character - ChoCh) แล้วหรือยัง?
- [ ] เข้าซื้อขายในโซนช่องว่างราคา (Fair Value Gap - FVG) หรือเข้าเมื่อราคาวิ่งกลับมาทดสอบ (Retest) หรือยัง?
- [ ] อัตราส่วนผลตอบแทนต่อความเสี่ยง (Risk-to-Reward Ratio - RRR) อยู่ที่อย่างน้อย **1:2** ขึ้นไปใช่หรือไม่?

### 3. จุดยอมแพ้ยกเลิกแผน (Stop Loss / Invalidation)
- แผนเทรดนี้จะล้มเหลวโดยสมบูรณ์ที่ราคาจุดไหน? ให้ตั้ง Stop Loss (SL) ห่างจากแนวโครงสร้างดังกล่าวด้านล่าง/ด้านบน 1-2 pips พอดี

- [ ] ฉันพร้อมยอมรับความสูญเสียในเทรดนี้ทั้งทางสภาวะการเงินและอารมณ์อย่างสมบูรณ์แบบแล้วใช่หรือไม่?
