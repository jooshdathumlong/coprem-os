## 1. Purpose
Defines: Axis-based product structure (FOCUS/CALM/PRESENT/RESET), production standards per axis, structural prohibitions for factories, "1 product = 1 state" interpretation, production-line quality standards.

## 2. Core Production Principle

| Rule | Detail |
|------|--------|
| **1 SKU = 1 Axis** | Each garment SKU maps to exactly 1 axis |
| **No mixing** | No cross-axis attributes in a single product |
| **Layering only** | Cross-axis mixing only via multi-layer wearing |
| **System, not fashion** | Factory must interpret products as "system" not "fashion" |

> Mixed interpretation = "system error" → must reproduce.

## 3. Axis Production Standard

### 3.1 FOCUS — The Armor System
**Goal:** Sharp silhouette, locked form, strong and still appearance.

| Spec | Requirement |
|------|------------|
| Fabric | Poly-Wool Blend or dense weave |
| GSM | 250–400 |
| Surface | Smooth, tight, minimal stretch |
| Pattern | Structured fit, sharp and still shoulders, no draping |
| Seam | Structural seam, sharp edge finish, high SPI |
| Hardware | All hidden, no visible metal |

**Forbidden:** ❌ Soft CALM fabrics ❌ Loose fits ❌ PRESENT textures

### 3.2 CALM — The Soft Shield System
**Goal:** Body relaxation, reduced pressure, natural drape.

| Spec | Requirement |
|------|------------|
| Fabric | Bamboo / Modal / Tencel |
| GSM | 140–190 |
| Surface | Soft, smooth, breathable |
| Pattern | Relaxed fit, no forced shaping, movement-friendly |
| Seam | Hidden or soft overlock, minimal friction |
| Hardware | Blended with fabric, no prominent metal |

**Forbidden:** ❌ Suit structure ❌ Rigid fabric ❌ Attention-grabbing details

### 3.3 PRESENT — The Expression System
**Goal:** Eye-catching, visible texture and detail, distinctive presence.

| Spec | Requirement |
|------|------------|
| Fabric | Linen blend / 3D knit / jacquard |
| GSM | 200–240 |
| Surface | Clear dimensional surface |
| Pattern | Semi-fitted, visual layering with focal points |
| Seam | Visible stitching / topstitch, visible craftsmanship |
| Hardware | Permitted at display level (metal / decorative) |

**Forbidden:** ❌ Flat/identity-less ❌ CALM minimal ❌ FOCUS hidden-everything

### 3.4 RESET — The Void System
**Goal:** Lightest possible, no structure, no fashion identity.

| Spec | Requirement |
|------|------------|
| Fabric | Muslin / airy gauze / light cotton |
| GSM | < 130 |
| Surface | Sheer, light, high breathability |
| Pattern | Very oversized, no shaping, no body definition |
| Seam | Flatlock only, minimize all seams |
| Hardware | None whatsoever |

**Forbidden:** ❌ Outerwear use ❌ Eye-catching details ❌ Any structure

## 4. Cross-Axis Control Rule

| Rule | Detail |
|------|--------|
| No material mixing | No cross-axis materials in single SKU |
| No pattern reuse | No borrowing patterns from other axes |
| No middle ground | No "mid-interpretation" between 2 axes |
| Single axis per SKU | Every SKU assigned exactly one axis |

## 5. Layering System

Factory does NOT produce "mixed outfits" — User assembles later.

```
Outer Layer:  FOCUS or PRESENT
Mid Layer:    CALM or PRESENT
Base Layer:   RESET or CALM
Private:      RESET only
```

**Key:** Each product is "pure" — mixing happens outside the factory only.
