## 1. CALM — คืนความสงบ สยบความเคลื่อนไหว

**Definition:** ความนิ่งที่ทรงพลัง (Powerful Stillness)

**The Logic:**
ไม่ใช่ความอ่อนแอหรือการพักผ่อนหย่อนใจ แต่คือการสร้างเกราะป้องกันทางอารมณ์
ในวันที่โลกภายนอกมีความเร็วสูงหรือเต็มไปด้วยความขัดแย้ง ระบบจะดึงอัตราการเต้นของหัวใจและความคิดที่ฟุ้งซ่านให้กลับมาอยู่ในจังหวะที่ควบคุมได้

**System Response:**
เมื่อเลือกแกนนี้ ระบบจะตัดสิ่งเร้า (Overstimulation) ทุกอย่างออก
- สิ่งที่สวมใส่จะไร้ความอึดอัด
- สิ่งที่บริโภคจะปรับสมดุลประสาท
- สภาพแวดล้อมจะถูกจัดสรรให้เหลือเพียงความเงียบสงบ
เพื่อให้คุณตัดสินใจเรื่องยากๆ ได้ด้วยความเยือกเย็น

## 2. FOCUS — เจาะจงเป้าหมาย ตัดสิ่งรบกวน

**Definition:** การพุ่งเป้าแบบไร้แรงเสียดทาน (Tunnel Vision)

**The Logic:**
การกำจัดตัวเลือกที่เหลือทั้งหมดเพื่อสิ่งเดียว
นี่คือแกนแห่งประสิทธิภาพสูงสุด สำหรับวันที่ต้องสร้างผลงานชิ้นเอก พรีเซนต์งานสำคัญ หรืองานที่ต้องการความแม่นยำระดับมิลลิเมตร
ความคิดลบและความลังเลจะถูกระงับชั่วคราว

**System Response:**
เมื่อเลือกแกนนี้ ระบบจะล็อกคุณเข้าสู่เป้าหมาย
- สิ่งที่สวมใส่จะมีฟังก์ชันเฉียบคมและพร้อมลุย
- สิ่งที่บริโภคจะกระตุ้นการทำงานของสมองขั้นสุด
- ระบบจะทำหน้าที่เป็นเกราะป้องกัน บล็อกผู้คนและการแจ้งเตือนที่ไม่จำเป็น

## 3. PRESENT — อยู่กับปัจจุบัน รับรู้ทุกสัมผัส

**Definition:** การเชื่อมโยงอย่างสมบูรณ์ (Deep Connection)

**The Logic:**
การดึงตัวเองออกจากอดีตที่ผ่านไปแล้วและอนาคตที่ยังมาไม่ถึง
เพื่อกลับมาหยั่งรากอยู่กับวินาทีตรงหน้า
สำหรับวันที่ต้องการใช้เวลากับครอบครัว ดื่มด่ำกับมื้ออาหาร หรือเสพประสบการณ์ที่ต้องใช้ความรู้สึกมากกว่าตรรกะ

**System Response:**
เมื่อเลือกแกนนี้ ระบบจะเปิดสวิตช์ประสาทสัมผัสทั้งห้า
- สิ่งที่สวมใส่จะถ่ายทอดความเป็นตัวตนและเปิดรับสัมผัส
- สิ่งที่บริโภคจะเน้นรสชาติและความรื่นรมย์
- ระบบจะช่วยให้คุณวางเครื่องมือสื่อสารลง และสบตากับคนตรงหน้าได้อย่างแท้จริง

---

## 4. RESET — ล้างกระบวนความจำ เริ่มต้นใหม่จากศูนย์

**Definition:** การล้างระบบเพื่อเริ่มใหม่ (System Reboot)

**The Logic:**
พื้นที่ว่างคือจุดเริ่มต้นของทุกสิ่ง
เมื่อผ่านความเหนื่อยล้าขั้นสุด จบโปรเจกต์ใหญ่ หรือรู้สึกว่าความคิดอิ่มตัว
นี่ไม่ใช่แค่การนอนหลับ แต่คือกระบวนการบังคับชัตดาวน์เพื่อล้างความทรงจำตกค้างและความเหนื่อยล้าสะสม

**System Response:**
เมื่อเลือกแกนนี้ ระบบจะสั่งให้ทุกอย่างหยุดนิ่ง
- สิ่งที่สวมใส่จะเน้นความปลอดภัยและให้ความรู้สึกเหมือนได้รับการปกป้อง
- สิ่งที่บริโภคจะมุ่งเน้นการฟื้นฟูระดับเซลล์
- ระบบจะกันคุณออกจากโลกภายนอก เพื่อเตรียมความพร้อมของร่างกายและสมองให้กลับมาเต็ม 100%

## Quick Reference

| Axis | State | System Action | Use When |
|------|-------|--------------|----------|
| **CALM** | Powerful Stillness | Cut overstimulation | World is chaotic, need clarity |
| **FOCUS** | Tunnel Vision | Lock onto one goal | Deep work, high-stakes delivery |
| **PRESENT** | Deep Connection | Open all senses | People, experiences, moments |
| **RESET** | System Reboot | Full shutdown & restore | After big projects, mental exhaustion |
