# PEABUNTID — Brand Knowledge Base

> 📌 Master index for all Peabuntid documentation. Maintained by Jeff. Updated: 2026-05-28

## Brand Identity
**Peabuntid** = "The Trusted Curator." A reliable information infrastructure for Thailand that curates, summarizes, and organizes data, products, and services.

## Core Philosophy
The world doesn't lack information — it lacks **reliable information**. Peabuntid is not a social platform, not an open marketplace, and not an agency. It is an **elder sibling who did the homework** and delivers only the best.

## 4 Core Systems

| System | Role |
|--------|------|
| **Knowledge Warehouse** | Verified, summarized knowledge articles |
| **Community Explorer** | Curated local database (shops/services) by geography |
| **Shop** | Pre-selected products and services only |
| **Tell the Scholar** | Submission channel for review (not instant publish) |
