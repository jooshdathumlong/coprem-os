# KOL Strategy — Eilinaire & Peadbuntid

> ที่มา: COPREM OS Internal | Updated: 2026-06-07

## KOL Tiers

| Tier | Followers | Rate (THB) | ใช้กับ |
|---|---|---|---|
| Nano | 1K-10K | 500-3,000 | Awareness + authenticity |
| Micro | 10K-100K | 3,000-20,000 | Conversion focus |
| Macro | 100K-1M | 20,000-200,000 | Brand building |
| Mega | 1M+ | 200,000+ | Awareness campaigns |

## Evaluation Criteria

1. **Engagement Rate** > 3% (nano), > 2% (micro), > 1% (macro)
2. **Audience match** — demographics ตรงกับ target
3. **Brand safety** — ไม่เคย endorse คู่แข่งโดยตรง
4. **Content quality** — aesthetic ตรงกับ brand DNA

## Brief Template

```
Brand: [Batiste/Eilinaire/Peadbuntid]
Campaign: [ชื่อ campaign]
Deliverables: [X TikTok, Y IG Story, Z Reels]
Key Message: [1 sentence]
Mandatory: [#hashtags, @mention, disclosure]
Don'ts: [สิ่งที่ห้ามพูด]
Deadline: [วันที่ส่ง draft / วันที่ post]
Payment: [amount, terms]
```

## Contract Checklist (Lex approval required)

- [ ] Usage rights (brand สามารถ repost ได้กี่วัน)
- [ ] Exclusivity (ห้าม endorse คู่แข่งกี่เดือน)
- [ ] Revision rounds (กี่รอบ)
- [ ] Late delivery penalty
- [ ] Disclosure requirement (#ad, #sponsored)

## ROI Calculation

```
KOL ROI = (Revenue attributed - KOL cost) / KOL cost × 100%
Attribution: UTM link หรือ promo code เฉพาะ KOL
Target: ROI > 150% = good deal
```
