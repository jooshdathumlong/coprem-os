
**Cowork —** **การทำงานร่วมกับคอมพิวเตอร์**

1. **ตรวจสอบโฟลเดอร์สื่อภาพตามแนวทางแบรนด์** — ให้ Claude อ่านไฟล์ภาพจำนวนมากเพื่อหาสีผิดแบรนด์ โลโก้เก่า หรือข้อความกฎหมายที่ขาดหาย
2. **ปรับหน้าตำราเรียนให้เหมาะกับทุกระดับการอ่าน** — แปลงหน้าตำราเรียนเดียวเป็นสไลด์และเอกสารอ่านสำหรับผู้เรียนหลายระดับ
3. **ควบคุมคอมพิวเตอร์จากระยะไกลด้วย** **Dispatch** — ส่งคำสั่งจากมือถือ ให้ Claude ทำงานบนคอมพิวเตอร์แทน
4. **เริ่มงานยาวจากแอปมือถือ** **Claude** — ตรวจสอบความคืบหน้าและสั่งงานต่อจากมือถือโดยไม่ต้องกลับไปที่โต๊ะ
5. **จัดการคำขอตอนไม่อยู่หน้าคอม** — Claude หาไฟล์ ร่างคำตอบ และรอให้อนุมัติก่อนส่ง
6. **ใช้งานแอปบนคอมจากมือถือผ่าน** **Dispatch** — ให้ Claude ควบคุมเมาส์และคีย์บอร์ดเพื่อทำงานในแอปที่ไม่มี API

  

**Finance —** **การเงิน**

1. **ร่างบันทึกสินเชื่อจากงบการเงินด้วย** **Claude for Excel** — ดึงข้อมูลผ่าน S&P Capital IQ แล้วอัปเดตโมเดลและเขียน credit memo
2. **ตรวจสอบเงินสำรองและร่างเอกสารยื่นด้วย** **Claude for Excel** — อ่านสมุดงานเงินสำรอง ตรวจสอบสูตร แล้วร่างบันทึกการยื่น
3. **อัปเดตโมเดลการเงินหลังประกาศผลประกอบการ** — ดึงรายงานและ transcript จาก S&P แล้วตรวจกับโมเดลการเงิน
4. **ทำความเข้าใจและต่อยอดสเปรดชีตที่ได้รับมา** — เข้าใจสูตรเดิมแล้วเพิ่มข้อมูลใหม่โดยรักษา logic เดิม
5. **กระทบยอดธุรกรรมข้ามบัญชี** — จับคู่ธุรกรรมจากหลายแหล่ง ตั้งค่าสถานะความคลาดเคลื่อน
6. **สร้างโมเดลการเงิน** — วิเคราะห์การลงทุนพร้อมวางแผนสถานการณ์และประเมินความเสี่ยง
7. **ร่างบันทึกการลงทุน** **(Investment Memo)** — สร้างจากข้อมูลแพลตฟอร์ม ตามรูปแบบของบริษัท
8. **จัดระเบียบการเงินธุรกิจ** — สร้างสเปรดชีตเพื่อดูเทรนด์และทำความเข้าใจตัวเลข

  

**Nonprofits —** **องค์กรไม่แสวงผลกำไร**

1. **ดู** **Theory of Change** **ในแชท** — อธิบายโปรแกรมแล้ว Claude วาดห่วงโซ่เหตุผล จากปัจจัยนำเข้าถึงผลกระทบ
2. **ดูว่าเป้าหมายแคมเปญต้องการอะไรจริง** **ๆ** — วาด gift pyramid แบบโต้ตอบ ลากเป้าหมายแล้วโครงสร้างจะปรับตาม
3. **เปรียบเทียบสถานการณ์งบประมาณ** — วาดสามสถานการณ์เคียงข้างกัน สลับระหว่างดอลลาร์กับเปอร์เซ็นต์
4. **ดูว่าทำไมการรักษาผู้บริจาคดีกว่าการหาผู้บริจาคใหม่** — สร้างการคาดการณ์ผู้บริจาค 5 ปีพร้อมตัวเลื่อน
5. **วางแผนปรับปรุงขั้นตอนการทำงาน** — ออกแบบโซลูชันที่ขับเคลื่อนด้วย AI เพื่อประหยัดเวลา
6. **สายการประกอบข้อเสนอขอทุน** — สร้างคลังเนื้อหาจากข้อเสนอที่สำเร็จแล้วผลิตเอกสารใหม่ได้เร็ว
7. **พัฒนาชุดเครื่องมือโปรแกรม** — สร้างกรอบแบบโปรแกรม logic model แผนประเมินผล
8. **แสดงข้อมูลโปรแกรมเป็นภาพ** — แปลงสเปรดชีตเป็นแผนภูมิ อินโฟกราฟิก และแดชบอร์ด
9. **สร้างระบบจัดการอาสาสมัคร** — เอกสาร onboarding แม่แบบการสื่อสาร เครื่องมือติดตาม
10. **วิเคราะห์ผลการระดมทุน** — วิเคราะห์ประสิทธิภาพข้ามช่องทาง หาจุดที่ให้ผลตอบแทนสูงสุด
11. **สร้างนโยบาย** **AI** — ครอบคลุมความเป็นส่วนตัว กรณีใช้งาน แนวทางพนักงาน จริยธรรม
12. **เขียนรายงานผลกระทบ** — เปลี่ยนข้อมูลดิบเป็นเรื่องเล่าพร้อม data visualization

  

**Education —** **การศึกษา**

1. **วางแผนหลักสูตร** — แนบ syllabus แล้ว Claude แสดงว่าสัปดาห์ไหนล็อกตาม prerequisite สัปดาห์ไหนจัดใหม่ได้
2. **พิจารณาตัวเลือกทุนในแชท** — Claude แสดงผู้ให้ทุนทั้งหมดในมุมมองเดียว กรองและจัดลำดับร่วมกัน
3. **ลองใช้สูตรขณะเรียนรู้** — สร้างกราฟ scatter แบบโต้ตอบในบทสนทนา วางจุดและลากดูผลลัพธ์
4. **ทำให้บทเรียนบนกระดานมีชีวิต** — Claude วาดภาพประกอบขณะวางแผนการสอน
5. **แสดงกลไกเบื้องหลังคำอธิบายเป็นภาพ** — สร้าง visual แบบโต้ตอบพร้อมปุ่มควบคุมขณะคุยกัน
6. **แมปงานทบทวนวรรณกรรมเพื่อหาข้อถกเถียง** — Claude อ่านงานวิจัยแล้ววาดโครงสร้างอาร์กิวเมนต์
7. **สร้างแผนภูมิข้อมูลก่อนตัดสินใจวิเคราะห์** — อัปโหลด CSV แล้ว Claude สร้าง correlation grid
8. **แปลงงานวิจัยเป็นงานนำเสนอ** — แปลงผลการวิจัยเป็นโครงร่างสไลด์และ speaker notes
9. **สร้างสื่อการเรียนรู้** — แปลงสมการและโน้ตลายมือเป็นเอกสาร LaTeX
10. **ฝึกสัมภาษณ์แบบ** **Case** **พร้อมฟีดแบ็ก** — ทำ case แบบ consulting พร้อม framework และฟีดแบ็ก
11. **วางแผนเส้นทางอาชีพ** — แมปงานที่ต้องการ ช่องว่างทักษะ ไทม์ไลน์ คนที่ควรติดต่อ

  

**Professional —** **การทำงานมืออาชีพ**

1. **สร้างสื่อแบรนด์** — นามบัตร ใบปลิว สื่อการตลาดตามแนวทางแบรนด์
2. **บรรจุแนวทางแบรนด์เป็น** **Skill** — สร้าง skill ที่ใช้สไตล์แบรนด์กับทุกงานอัตโนมัติ
3. **สำรวจความสามารถของ** **Claude** — บอกบทบาทแล้วได้คำแนะนำส่วนตัว
4. **สร้างการวิเคราะห์จากแผนภูมิเบราว์เซอร์และข้อมูลโฟลเดอร์** — ดึงข้อมูลจากหลายแหล่งมาเปรียบเทียบ
5. **สร้างสรุปประจำวันจากเครื่องมือต่าง** **ๆ** — ดึงจาก Slack, Notion, แดชบอร์ดทีม
6. **ประมวลผลผู้ขายเป็นชุดด้วย** **Cowork** — อ่านไฟล์ผู้ขาย เพิ่มใน tracker สร้างสัญญา กรอกแบบฟอร์ม
7. **ประเมินขนาดตลาดจากงานวิจัย** — วิจัย คำนวณ และส่งออก PowerPoint, Excel, Markdown
8. **ดึงข้อมูลเชิงลึกจากเครื่องมือมาสร้างเดค** — ค้นหาข้ามแหล่งแล้วสร้างงานนำเสนอ
9. **ประเมินบริษัทจากวิทยาศาสตร์ถึงงบดุล** — Due diligence ข้าม SEC filings, clinical trial, สิทธิบัตร
10. **สร้างรายงานสถานะโครงการ** — ดึง update จากอีเมล Slack ประชุม มาสร้าง tracker
11. **วิเคราะห์รูปแบบในฟีดแบ็กผู้ใช้** — หาธีมซ้ำ ๆ และจุดปวดจากฟีดแบ็ก
12. **เตรียมตัวสำหรับสัปดาห์อย่างรวดเร็ว** — จัดลำดับความสำคัญจากปฏิทินและอีเมล
13. **เปรียบเทียบและวิเคราะห์ตัวเลือก** — อัปโหลดข้อเสนอผู้ขาย สร้างตารางเปรียบเทียบ
14. **สร้างฟอร์ม** **PDF** **แบบโต้ตอบ** — แปลงฟอร์มเอกสารเป็นฟอร์มกรอกได้
15. **สร้างผังกระบวนการ** — แปลงขั้นตอนเป็น flowchart
16. **แปลงอีเมลเป็นตัวติดตามกิจกรรม** — ดึงวันที่ สถานที่ รายละเอียดจากอีเมล
17. **สร้างจดหมายข่าวบริษัท** — สรุปข่าวจากทุกช่องทางแบบ publication-style

  

**Personal —** **ใช้ส่วนตัว**

1. **สร้างเว็บเพจ** — สร้างเว็บพอร์ตโฟลิโอโดยไม่ต้องเขียนโค้ด
2. **ยกระดับการออกแบบด้วย** **Skills** — สร้าง skill ที่ใช้หลักการออกแบบอัตโนมัติ
3. **สร้างเครื่องมือแผนภาพแบบโต้ตอบ** — แปลง prompt เป็น reference app
4. **จัดระเบียบไฟล์บนเดสก์ท็อป** — ให้ Cowork อ่านและจัดเรียงไฟล์เป็นโฟลเดอร์
5. **แมปความเข้าใจและสร้างบทเรียนจากช่องว่าง** — หาจุดเข้าใจผิดและสร้างประสบการณ์เรียนรู้ส่วนตัว
6. **ทดสอบแผนการเงินหลายสถานการณ์** — สร้างโมเดลที่ปรับได้เพื่อดูผลกระทบลูกโซ่
7. **ให้ของขวัญอย่างมีใจ** — แนะนำของขวัญ ค้นหาร้านค้า ประสานการซื้อ
8. **แปลงข้อความเป็นโน้ตพร้อมข้อมูล** — ค้นหาข้อความ วิจัยคำตอบ บันทึกลง Notes
9. **สร้างรายการ** **Bucket List** — สร้างเครื่องมือติดตามเป้าหมายแบบโต้ตอบ
10. **ออกแบบคู่มือหาของป่า** — แผนที่แบบโต้ตอบ เรียกดูตามหมวด พิมพ์ได้
11. **สร้างโน้ตสุขภาพและการออกกำลังกาย** — ค้นคว้าท่าออกกำลังกายบันทึกลง Notes
12. **สร้างการ์ดสูตรอาหารดิจิทัล** — แปลงสูตรลายมือเป็นดิจิทัล
13. **สร้างแผนเที่ยวรายวัน** — ตารางเที่ยวปรับตามความชอบ
14. **ฝึกดีเบตพร้อมฟีดแบ็ก** — ทดสอบไอเดียกับมุมมองตรงข้าม
15. **วิจัยและเปรียบเทียบจุดหมายท่องเที่ยว** — สเปรดชีตเปรียบเทียบพร้อมรูปและเรตติ้ง
16. **แปลงแรงบันดาลใจเป็นแผนออกแบบ** — แปลงรูปแรงบันดาลใจเป็นแผนปรับปรุงบ้าน

  

**Legal —** **กฎหมาย**

1. **ตรวจสอบสัญญาและเจรจา** **(Redlining)** — หาเงื่อนไขที่มีผลกระทบ พร้อมข้อเสนอแนะ
2. **เตรียมเอกสารสำหรับการตรวจสอบ** **Compliance** — จัดเรียงเอกสารนโยบาย สัญญา และบันทึก
3. **จัดระเบียบงานกฎหมายด้วย** **Projects** — อัปโหลด playbook แล้วอ้างอิงอัตโนมัติ
4. **ติดตามไทม์ไลน์** **Discovery** **และวิเคราะห์รูปแบบ** — สร้างลำดับเหตุการณ์และหารูปแบบเอกสาร

  

**Sales —** **การขาย**

1. **สร้างงานนำเสนอข้อเสนอขาย** — เดคพร้อม data visualization และดีไซน์
2. **เตรียมตัวสำหรับดีลขาย** — ดึงข้อมูล CRM เปรียบเทียบโอกาสที่คล้ายกัน
3. **สร้างรายงานการขาย** — ดึง metrics จาก CRM วิเคราะห์เทรนด์ สร้างรายงาน
4. **สร้างคลัง** **Battle Card** — แปลงข้อมูลคู่แข่งเป็น battle card พร้อม talk track

  

**Claude in Chrome —** **ส่วนขยายเบราว์เซอร์**

1. **บันทึกสายขายเข้า** **CRM** — อ่านปฏิทิน จับคู่ผู้เข้าร่วมกับ Salesforce ร่าง activity log
2. **เปรียบเทียบสินค้าข้ามเว็บไซต์** — อ่าน specs จากหลายแท็บ สร้างตารางเปรียบเทียบใน Google Sheets
3. **เตรียมตัวและวางแผนจากปฏิทิน** — อ่านปฏิทิน ดึงบริบทจากอีเมล จองห้องประชุม
4. **จัดการอีเมลโปรโมชัน** — สแกน inbox หาอีเมลโปรโมชันเพื่อลบเป็นชุด
5. **ดึง** **metrics** **จากแดชบอร์ด** **Analytics** — นำทางแดชบอร์ด ดึงตัวเลข รวบรวมเป็นสรุป
6. **จัดระเบียบไฟล์ใน** **Google Drive** — สร้างโครงสร้างโฟลเดอร์ ย้ายไฟล์ ตั้งค่าสถานะไฟล์ซ้ำ

  

**Marketing —** **การตลาด**

2. **วิเคราะห์ผลแคมเปญ** — หาช่องทางที่ดีที่สุด/แย่ที่สุด พร้อมคำแนะนำจัดสรรงบ
3. **สร้าง** **Customer Personas** — สร้างจาก demographics เป้าหมาย และ pain points

  

**Research —** **การวิจัย**

1. **สกัดธีมจากช่องทางฟีดแบ็กทั้งหมด** — สังเคราะห์จาก call transcripts, Slack, CRM, Linear
2. **แปลงเวลาเดินทางเป็นเวลาวิจัย** — บันทึกความคิดด้วยเสียงบนมือถือ แล้วให้ Claude วิจัยต่อ
3. **ตรวจสอบสถิติจากข้อมูลดิบ** — ตรวจสอบสถิติที่เผยแพร่กับข้อมูลต้นฉบับ
4. **วางแผนทบทวนวรรณกรรม** — หางานวิจัยที่เกี่ยวข้อง จัดลำดับความสำคัญ จัดระเบียบหลักฐาน

  

**Life Sciences —** **วิทยาศาสตร์ชีวภาพ**

1. **วิเคราะห์การศึกษาก่อนคลินิก** — สร้างรายงานจากแพลตฟอร์มวิจัย รวบรวมข้อมูลข้ามการทดลอง
