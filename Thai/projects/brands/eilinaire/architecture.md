## 1. BRAND POSITIONING

Eilinaire ไม่ใช่แบรนด์สินค้า แบรนด์แฟชั่น หรือแอปพลิเคชัน
แต่คือ **Layer ที่อยู่เหนือสินค้า** ทำหน้าที่เป็นตัวกรองความวุ่นวาย
เพื่อให้คุณเลือกตัวเองได้ชัดเจนขึ้นในแต่ละวัน

| Layer | Definition |
|-------|-----------|
| **Logical Layer** (สิ่งที่แบรนด์เป็น) | Decision System for Everyday Life |
| **Emotional Layer** (สิ่งที่ผู้ใช้รู้สึก) | Less choice. More clarity. |

## 2. THE CORE ARCHITECTURE

ระบบถูกออกแบบมาอย่างจำกัดและสมบูรณ์แบบในตัวเองผ่าน **4 แกนหลัก**
**ล็อกไว้ที่ 4 แกนเท่านั้น — ไม่มีการเพิ่ม ไม่มีการแตกย่อย**

- **CALM**
- **FOCUS**
- **PRESENT**
- **RESET**

## 3. SYSTEM LOGIC & FLOW

เปลี่ยนจุดเริ่มต้นของการใช้ชีวิต:
จาก → *"วันนี้จะใส่อะไร"* หรือ *"จะกินอะไร"*
เป็น → **"วันนี้ฉันจะเป็นแบบไหน"**

| Step | Action |
|------|--------|
| **Step 1: Choose** | ผู้ใช้เลือก 1 แกนหลักสำหรับวันนั้น |
| **Step 2: System Response** | ระบบจัดสรรโซลูชันที่สอดคล้อง (สิ่งที่ใส่ กิน และใช้ชีวิต) ทันที |
| **Step 3: Alignment** | ผู้ใช้เข้าสู่ความรู้สึกและโฟกัสในแกนนั้นตลอดทั้งวัน |

## 4. THE PRODUCT ECOSYSTEM

สินค้าทุกชิ้นไม่ใช่การขายแยกชิ้น แต่เป็นส่วนหนึ่งของระบบที่ตอบสนองต่อแกนหลักที่ถูกเลือก:

| Line | Description |
|------|------------|
| **Eilinaire FUEL** | ปรับสมดุลจากภายใน (โภชนาการ เครื่องดื่ม อาหาร) |
| **Eilinaire LOOK** | สะท้อนภาพลักษณ์และสร้างความรู้สึกภายนอก (เสื้อผ้า เครื่องแต่งกาย) |
| **Eilinaire LIFE** | รองรับการใช้ชีวิตระหว่างวัน (เครื่องมือ สภาพแวดล้อม สิ่งของเครื่องใช้) |
| **Eilinaire PASS** | กำหนดกระแสเวลาและการเข้าถึง (การจัดสรรเวลา สิทธิพิเศษ พื้นที่) — *Future* |

## 5. VALUE PROPOSITION & TARGET AUDIENCE

### Core Value
**Less choice. More clarity.**

### Value Dimensions
| Type | Value Delivered |
|------|----------------|
| **Functional** | ลดภาระการตัดสินใจ (Cognitive Load) — เปลี่ยนชีวิตให้เป็นระบบอัตโนมัติที่มีประสิทธิภาพ |
| **Emotional** | มอบความนิ่ง ความมั่นใจ และความชัดเจนขั้นสุด |
| **Identity** | "ฉันรู้ว่าฉันกำลังเป็นใครและทำอะไรอยู่" |

### Target Customer
- **Primary:** Founder, Creator และคนทำงานที่ชีวิตมีความเร็วสูง แต่อยากได้ "ความชัดเจน"
- **Mindset:** เบื่อหน่ายการเลือกยิบย่อย ต้องการลดตัวเลือกในชีวิต และพร้อมใช้ระบบที่ถูกคิดมาให้แล้ว

## 6. COMPETITIVE EDGE & BRAND RULES

### จุดได้เปรียบทางธุรกิจ
- **Constraint as Premium** — การมีข้อจำกัดคือความพรีเมียม เรามีเพียง 4 ตัวเลือกเท่านั้น
- เราขาย **"ระบบ"** ที่มีตรรกะการใช้งานจริง ไม่ใช่ขายคอนเซปต์ลอยๆ หรืออารมณ์แบบแบรนด์แฟชั่น

### Brand Rules (กฎเหล็ก)
1. ทุกสิ่งต้องเริ่มต้นจาก 4 แกนหลัก — ห้ามสื่อสารหรือขายสินค้าโดยปราศจากบริบทนี้
2. ภาษาและการออกแบบต้อง **เรียบ แต่คม** และทำให้ผู้ใช้รู้สึกได้ทันทีตั้งแต่แรกเห็น

## 7. REVENUE & DISTRIBUTION MODEL

### โครงสร้างรายได้
| Stream | Description |
|--------|------------|
| **Direct Product** | ยอดขายจากผลิตภัณฑ์ในระบบ (FUEL, LOOK, LIFE) |
| **System Access (Future)** | Subscription model เพื่อเข้าถึงสิทธิประโยชน์และการจัดการชีวิต (PASS) |
| **Experience** | พื้นที่และสภาพแวดล้อมที่จัดสรรมาเฉพาะ (Curated Space) |

### การกระจายสินค้า
| Phase | Channel |
|-------|---------|
| **Phase 1** | Direct-to-Consumer ผ่านหน้าเว็บไซต์แบบ One-page |
| **Phase 2** | Pop-up Store เพื่อสร้างประสบการณ์จริง |
| **Phase 3** | เชื่อมต่อระบบเข้ากับพาร์ทเนอร์ต่างๆ |

## 8. GO-TO-MARKET STRATEGY (ACTIONABLE)

| Step | Action |
|------|--------|
| **Step 1** | ล็อกเป้าหมายเปิดตัวที่แกน **FOCUS** เพียงแกนเดียว |
| **Step 2** | สร้างผลิตภัณฑ์จริง 2 ชิ้น: เครื่องดื่ม (FUEL) 1 ตัว + เสื้อ (LOOK) 1 ตัว |
| **Step 3** | ออกแบบหน้าเว็บ One-page: เลือกเป้าหมาย → แสดงผลลัพธ์ → สั่งซื้อทันที |
| **Step 4** | ทดสอบกับกลุ่มผู้ใช้จริงเพื่อพิสูจน์ระบบ |

## 9. THE ULTIMATE VISION

> เป้าหมายของ Eilinaire ไม่ใช่การเป็นเพียงแบรนด์ไลฟ์สไตล์

>
> แก่นแท้ของแบรนด์ไม่ใช่การขายสินค้า
> แต่คือการ **ลบความวุ่นวายออกจากชีวิตมนุษย์**
> คืนเวลาและศักยภาพกลับไปสู่สิ่งที่สำคัญที่สุด
