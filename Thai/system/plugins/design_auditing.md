# ปลั๊กอินการตรวจสอบการออกแบบ (Design Auditing Plugin)
(ระบบเสริมขีดความสามารถภายใน)

**ขีดความสามารถ:**
- การตรวจสอบความสอดคล้องของ UI/UX เทียบกับรัฐธรรมนูญแบรนด์ (Brand Constitution)
- การตรวจสอบความเปรียบต่างของสี (Color Contrast) และมาตรฐานการเข้าถึง (Accessibility Web Standards) โดยอัตโนมัติ
- การดึงโครงสร้างเลย์เอาต์จาก Figma และการตรวจสอบความถูกต้อง
